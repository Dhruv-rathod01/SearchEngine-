# 🔍 SearchEngine AI

A modern AI-powered image search engine developed using Java JSP/Servlet technology with dynamic image fetching and a responsive UI.

---

# 🚀 Features

- Dynamic image search functionality
- Real-time image rendering
- Modern responsive UI/UX
- Multiple image results using JSP loops
- Responsive image gallery
- API integration for live image fetching
- Apache Tomcat deployment
- Smooth modern interface

---

# 🛠️ Tech Stack

- Java
- JSP
- Servlet
- HTML
- CSS
- JavaScript
- Apache Tomcat
- LoremFlickr API

---

```text
ServletProject

├── src

    │
```

---

# ⚡ How It Works

1. User enters a search keyword.
2. JSP processes the request.
3. Dynamic image URLs are generated using API integration.
4. JSP loops dynamically render multiple images.
5. Images are displayed in a responsive gallery layout.

# ▶️ Run Locally

## 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/SearchEngine-AI.git
```

---

## 2. Place Project in Apache Tomcat

Copy the project folder into:

```text
C:\apache-tomcat-9.0.117\webapps
```

---

## 3. Start Apache Tomcat

```bash
cd C:\apache-tomcat-9.0.117\bin
startup.bat
```

---

## 4. Open in Browser

```text
http://localhost:8080/ServletProject/webapp/index.jsp
```
